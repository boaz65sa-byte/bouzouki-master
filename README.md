# 🎸 Bouzouki Master - אפליקציית לימוד בוזוקי מקצועית

אפליקציה אינטראקטיבית ללימוד מודוסים יווניים (דרומוס) ואקורדים על בוזוקי טטראכורדו בן 4 מיתרים.

## ✨ תכונות

- 🎼 **7 מודוסים יווניים מלאים** - Rast, Hitzaz, Ousak, Pireotikos, Niavent, Sambah, Kiourdi
- 🎹 **8 אקורדים בסיסיים** - Major, Minor, Dominant 7 ועוד
- 🎨 **מצב לילה/יום** - ממשק מותאם לכל תנאי תאורה
- 📱 **עיצוב רספונסיבי** - עובד מצוין במחשב, טאבלט ומובייל
- ⚡ **ביצועים מהירים** - נבנה עם React + Vite
- 🌍 **ממשק בעברית** - כולל תמיכה מלאה בכיוון RTL

## 🚀 התקנה והרצה

### דרישות מוקדמות
- Node.js (גרסה 16 ומעלה)
- npm או yarn

### הוראות התקנה

```bash
# 1. התקן תלויות
npm install

# 2. הרץ בסביבת פיתוח
npm run dev

# 3. בנה לפרודקשן
npm run build

# 4. תצוגה מקדימה של גרסת הפרודקשן
npm run preview
```

האפליקציה תיפתח אוטומטית בדפדפן שלך בכתובת: `http://localhost:3000`

## 📁 מבנה הפרוייקט

```
bouzouki-app/
├── src/
│   ├── components/          # קומפוננטות React
│   │   ├── Header.jsx       # כותרת וניווט
│   │   ├── Fretboard.jsx    # תצוגת צוואר הבוזוקי
│   │   ├── ControlPanel.jsx # פאנל בקרה למודוסים ואקורדים
│   │   └── InfoPanel.jsx    # פאנל מידע
│   ├── data/
│   │   └── musicData.js     # נתוני מודוסים ואקורדים
│   ├── App.jsx              # קומפוננטת ראשית
│   ├── main.jsx             # נקודת כניסה
│   └── index.css            # סגנונות גלובליים
├── index.html               # HTML ראשי
├── package.json             # תלויות ופקודות
└── vite.config.js           # הגדרות Vite
```

## 🎯 שימוש באפליקציה

1. **בחר מוד או אקורד** - לחץ על אחד הכפתורים בפאנל הבקרה
2. **צפה בצוואר** - התווים יסומנו אוטומטית על הצוואר
3. **זהה טוניקה** - תווים אדומים הם הטוניקה (Root)
4. **תרגל** - עקוב אחר התווים הזהובים
5. **נקה** - לחץ על "נקה את הצוואר" כדי להתחיל מחדש

## 🎨 התאמה אישית

ניתן לשנות צבעים ב-`src/index.css`:

```css
:root {
  --gold: #d4af37;          /* צבע זהב ראשי */
  --highlight: #ff4500;      /* צבע טוניקה */
  --bg-dark: #1a1a1a;       /* רקע כהה */
  /* ... */
}
```

## 📦 בניה לפרודקשן

```bash
npm run build
```

הקבצים המוכנים להעלאה יהיו בתיקיית `dist/`

### העלאה לשרת

אפשר להעלות את תוכן תיקיית `dist/` לכל שרת סטטי:
- **Netlify** - גרור ושחרר את תיקיית dist
- **Vercel** - חבר את הריפוזיטורי ב-GitHub
- **GitHub Pages** - השתמש ב-GitHub Actions
- **שרת אישי** - העתק את הקבצים לתיקיית ה-public
